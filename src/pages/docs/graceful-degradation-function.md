---
title: Graceful Degradation Function
description: About Graceful Degradation Function primitive.
---

Function that accepts different variants of the same UI component known as the [Carbon Component](/docs/carbon-component). It returns the one that is associated with the chosen Display Mode. These components share the same core functionality but shall operate with a different carbon efficiency.
