---
title: Graceful Degradation Function
description: Function that accepts different variants of the same UI component.
---

Function that accepts different variants of the same UI component known as [Carbon Component](/docs/carbon-component). It returns the one that is associated with a chosen display mode. These components share the same core functionality but shall operate with a different carbon efficiency.
